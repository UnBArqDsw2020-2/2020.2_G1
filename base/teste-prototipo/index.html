Metodologia


Para a realização dos testes dos protótipos de alta fidelidade foi utilizado o Whatsapp para receber feedback de possíveis usuários da plataforma acerca do protótipo feito. Tal teste tem como objetivo analisar a partir da opinião dos participantes se o protótipo tem algum problema de usabilidade ou se alguma funcionalidade não é de relevância do escopo.

Resultado






De acordo com os testes o protótipo é válido, suas funcionalidades estão de acordo com o que um usuário esperaria do sistema e o design é adequado.
